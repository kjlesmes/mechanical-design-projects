# Mechanical Design Projects

This repository contains a collection of my mechanical engineering design projects.

## Projects

### Quadcopter drone
Mechanical design project of a quadcopter drone, including CAD modeling, technical drawings and structural analysis through fall simulation.

Repository:
https://github.com/kjlesmes/Dron-Cuadriocoptero

---

### Robotic Arm CAD
CAD design project for a miniature robotic arm developed in SolidWorks. The system consists of a fixed base and three articulated links that make up the manipulator's kinematic structure.

Repository:
https://github.com/kjlesmes/Brazo-Mecanico

---

### Weather Station
CAD design of a weather station consisting of an electronic control box and a Stevenson cage for the protection of environmental sensors.

Repository:
https://github.com/kjlesmes/Estacion-meteorologica.

---

### Hysteresis filler
Liquid filling system based on hysteresis control. This project includes the complete mechanical design of an automated device for the precise filling of containers using a rotary plate mechanism.

Repository:
https://github.com/kjlesmes/Llenador-por-Histeresis

---

### Bench Vise
Mechanical design project for a bench vise developed in SolidWorks. The assembly includes all the components necessary for the complete operation of the clamping mechanism.

Repository:
https://github.com/kjlesmes/Prensa-de-Banco-

---

### Juicer CAD 3D
Mechanical design project for a juicer developed in SolidWorks. Includes complete part modeling, assembly, and technical manufacturing drawings.

Repository:
https://github.com/kjlesmes/Exprimidor-CAD3D

---

### Air Filter Mitsubishi Lancer GSR 1981
This project contains the 3D CAD design and technical drawings for an air filter component compatible with the 1981 Mitsubishi Lancer GSR.

Repository:
https://github.com/kjlesmes/Filtro-de-Aire---Mitsubishi-Lancer-GSR-1981
